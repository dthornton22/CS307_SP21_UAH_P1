This zip file contains 9 files:

1. ReadMe.txt - The text file you are reading now.

2. FlightSim01.txt - A data file listing the two files containing data on cities and flights.

3. CityData01.xml - A data file defining all of the cities in this version of the simulation with
	the distances between the cities.

4. AirlineFlightData01.xml - A data file defining all the aircraft and flights in the scenario. 

5. CityDataParser.h - The header file for a parser for the city data file

6. CityDataParser.cpp - The implementation file for a class that will handle reading and parsing 
	the city data file.  Look at the functions prototyped in the header file for details on 
	how it works.

7. FlightDataParser.h - The header file for a parser for the airline flight data file.

8. FlightDataParser.cpp - The implementation file for a class that will handle reading and parsing 
	the airline flight data file.  Look at the functions prototyped in the header file for details 
	on how it works.

9. Prog1.exe - This is an executable file containing a demonstration
	of programming assignment 1. Run this to see an example implementation showing how the program
	should behave.
